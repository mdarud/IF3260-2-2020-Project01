Halaman Awal

Halaman drawing canvas ini memiliki tombol-tombol untuk memilih warna apa saja, me-reset canvas dan menggambar 3 bentuk pilihan (LINE, SQUARE & POLYGON)


Draw Mode / Reset Canvas

Sebelum menggambar di canvas, haruslah menekan tombol "Draw!" terlebih dahulu.

Ketika mau mengosongkan canvas, bisa juga dengan menekan tombol "Draw!"


Line Mode

Ketika ingin berubah ke mode menggambar garis, bisa dengan menekan tombol "DrawLines"


Line Mode Sketching

Prekonsidi sudah berada di mode menggambar garis. Untuk menggambar garis, bisa dengan menekan mouse di dua posisi yang berbeda maka akan muncul gambar garis


Polygon Mode

Ketika ingin berubah ke mode menggambar poligon, bisa dengan menekan tombol "DrawPoligon"


Polygon Mode Sketching

Prekonsidi sudah berada di mode menggambar poligon. Untuk menggambar poligon, bisa dengan menekan mouse di beberapa posisi maka akan muncul banyaknya garis berdasarkan banyaknya tekan mouse


Square Mode

Ketika ingin berubah ke mode menggambar mode, bisa dengan menekan tombol "DrawSquare"


Square Mode Sketching

Prekonsidi sudah berada di mode menggambar persegi. Untuk menggambar persegi, bisa dengan menekan mouse di dua posisi yang berbeda maka akan muncul gambar persegi


Change Color

Ketika ingin berubah warna menggambar, bisa dengan menekan salah satu dari pilihan warna
